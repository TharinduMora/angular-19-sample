RXJS

RXJS is a library that helps you work with data that changes over time, like user actions, web requests, or timers. It makes handling these updates easier and more organized.

In Angular, RXJS is often used for things like handling events and making HTTP requests. It helps you write code that is easier to read and manage by letting you change and combine data streams in simple ways.

Async Pipe

The Async Pipe is a built-in Angular pipe that allows you to subscribe to an Observable or Promise and automatically unsubscribe when the component is destroyed.

    Usage of interval , switchMap & filter operators

    interval creates an observable that emits sequential numbers every specified interval (in milliseconds)

    switchMap is used to switch to a new observable when the source emits a value

    filter is used to filter emitted values based on a condition
